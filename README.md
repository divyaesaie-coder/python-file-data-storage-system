# Python File Data Storage System

A beginner-friendly Python project that stores and retrieves user information using file handling.

## Features
- Store user details permanently
- Retrieve saved data from file
- Uses text file storage
- Menu-driven console application

## Technologies Used
- Python 3
- File Handling

## How It Works
The program allows users to:
1. Store user details like name, age, and salary
2. Save the data into a text file
3. Retrieve and display stored data
4. Exit the system

## Concepts Used
- File handling
- Reading and writing files
- while loop
- if-elif conditions
- String manipulation

```bash
